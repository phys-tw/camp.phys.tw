＊新特性
   TCBB 網站統計系統 版本 2.0使用文本方式統計記錄數據，無需資料庫系統支持，通用性更強

＊安裝環境
   php4.1.0以上
   TCBB Stat System只在上述環境測試安裝成功, 其他環境不保證能夠安裝成功。

＊安裝說明
    1、修改 stat/config/config.php 檔案，進行個性化設置。
        注意：請修改$conf['stat_url']以和您的統計系統實際URL統一 。
              請修改管理員名稱和密碼
                $conf['admin_username']
                $conf['admin_password']

    2、將stat/資料夾上傳至您的伺服器指定目錄中，例如Web根目錄,上傳後則為/stat 。

    3、請修改下列資料夾的權限屬性為可寫（777）
         /stat/data (在執行安裝程式前必須設定)

    4、在瀏覽器中執行 http://yourhostname/stat/setup/index.php
        該安裝精靈將會引導您完成安裝。


＊嵌入代碼使用辦法
    在安裝精靈運行的最後，它會告訴您如何嵌入代碼到您想進行統計的頁面。
    如果您沒有及時記下它提示的代碼，請參考以下說明。

    注意：請將下面代碼中的 http://www.bbsfans.com/stat 更改為您的統計系統的實際URL

    <script language="Javascript">
        var stat_url = "http://www.bbsfans.com/stat";
    </script>
    <script language="Javascript" src="http://www.bbsfans.com/stat/stat.js"></script>
    <script language="Javascript" src="http://www.bbsfans.com/stat/display.php"></script>
